# About Me

Hello, I'm Prajwal.

I'm a Cloud Security Engineer with experience in cloud infrastructure, platform engineering, DevOps, and security automation. Over the years, I've worked across AWS, Terraform, Kubernetes, Python, CI/CD platforms, and cloud security tooling.

My professional journey has taken me through infrastructure engineering, cloud operations, platform engineering, and, more recently, cloud security. Today, much of my work focuses on Cloud Security Posture Management (CSPM), security automation, policy enforcement, compliance visibility, and building scalable solutions that help engineering teams operate securely in the cloud.

I started this blog as a place to organize and share technical knowledge that I've accumulated throughout my career. Like many engineers, I have countless notes, experiments, troubleshooting guides, and partially written documents spread across different tools and systems. This site is my effort to bring them together in a single place for future reference and, hopefully, to help others facing similar challenges.
